Apache, Mac, Server-side

Webサイトを作る場合、直接ブラウザでファイルを開くのではなく、サーバー内に設置したファイルをhttp通信で取得して、ブラウザで確認します（先輩の受け売り）。

てことで、手元のMacにローカルサーバを作って、それ経由でウェブコンテンツを確認できる環境を作ります。

Apache HTTP server

Apacheの設定ファイル

Apacheの設定ファイルは、Apacheのディレクトリ内の「httpd.conf」。

このファイルはテキストファイルであり、ディレクティブ（Apacheを設定するコマンドを意味する用語）がたくさんコメントアウトされたものも含め書かれています。

このファイルには重要な項目が沢山らしいので、別記事でフォーカスをあてます。

とりあえず今回チェックするのはドキュメントルートの場所。localhostで接続したIt worksのページがどこに存在しているかを調べます。

「/Library/WebServer/Documents」だそうです。

Apacheを使って”（・８・）ホノカチャン！”
